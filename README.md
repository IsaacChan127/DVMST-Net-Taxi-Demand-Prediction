# DVMST-Net-Taxi-Demand-Prediction
This project implements a deep learning–based taxi demand forecasting system using a spatio-temporal neural network architecture inspired by DVMST-Net. The model predicts short-term hourly taxi pickup demand for individual New York City taxi zones using historical demand patterns and temporal contextual features.

The implementation focuses on zone-level modelling, where each taxi zone is trained independently using historical demand sequences.
